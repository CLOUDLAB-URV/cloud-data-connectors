Install & Configuration
=======================


.. note::

   `Dataplug` is in active development: its APIs and functionality are subject to change!

   Please update your installation regularly in order to get the latest features, bug fixes and code improvements.


You can install `Dataplug` using ``pip``:


.. code-block:: bash

    pip install cloud-dataplug


or by cloning the ``master`` branch from Github:

.. code-block:: bash

    git clone https://github.com/CLOUDLAB-URV/dataplug
    cd dataplug
    pip install -e .

.. warning::

   Installing from ``master`` branch is for development purposes and is unstable.
